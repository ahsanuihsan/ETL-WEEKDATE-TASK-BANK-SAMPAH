Query 1: Membuat Class untuk Sistem UMKM

Kita mulai dengan membuat class UMKMSystem yang akan menjadi inti dari sistem ini. Class ini akan menyimpan data dasar seperti nama UMKM, data anggota, dan dana pinjaman awal dari Bapak Arry Hutomo.

Tip Kunci:

Do: Berikan nama atribut yang jelas dan mudah dipahami, seperti nama_umkm. Pastikan setiap atribut diinisialisasi dengan benar di awal.

Don't: Jangan lupa inisialisasi anggota sebagai daftar kosong; ini mencegah error saat data anggota mulai ditambahkan.


class UMKMSystem:
    def __init__(self, nama_umkm):
        self.nama_umkm = nama_umkm
        self.anggota = []
        self.dana_pinjaman = 50_000_000  # Rp50 juta

Hasil: Definisi kelas ini mempersiapkan "cetak biru" UMKM, siap untuk diisi dengan informasi.

Query 2: Method untuk Menambah Anggota UMKM

Menambahkan method tambah_anggota untuk menyimpan data anggota (nama dan pinjaman) dalam list anggota.Method ini menerima nama dan jumlah_pinjaman, membuat dictionary, dan menambahkannya ke list anggota . Pesan konfirmasi ditampilkan dengan format rupiah.

Tip Kunci:

Do: Gunakan dictionary untuk setiap anggota; ini membuat data mudah diakses dan dicari. Gunakan format angka yang bersih (:,) untuk keterbacaan yang optimal.

Don't: Hindari menyimpan data anggota hanya sebagai list sederhana tanpa struktur dictionary. Karena data memiliki konsep Key - Value


    def tambah_anggota(self, nama, jumlah_pinjaman):
        anggota = {"nama": nama, "pinjaman": jumlah_pinjaman}
        self.anggota.append(anggota)
        print(f"Anggota {nama} ditambahkan dengan pinjaman Rp{jumlah_pinjaman:,}")

Contoh Aksi:

Anggota Ihsan ditambahkan dengan pinjaman Rp5,000,000

Query 3: Method untuk Menghitung Pengembalian Pinjaman

Metode ini memungkinkan untuk menghitung proyeksi total pengembalian pinjaman seorang anggota, lengkap dengan bunga sederhana 5% per tahun.

Tip Kunci:

Do: Manfaatkan loop untuk mencari anggota secara efisien. Format output dengan :, .2f untuk menampilkan nilai mata uang yang rapi dengan dua desimal.

Don't: Jangan pernah berasumsi anggota yang dicari pasti ada. Selalu sertakan validasi untuk memberikan pesan jika anggota tidak ditemukan.


    def hitung_pengembalian(self, nama, tahun):
        for anggota in self.anggota:
            if anggota["nama"] == nama:
                bunga = anggota["pinjaman"] * 0.05 * tahun
                total = anggota["pinjaman"] + bunga
                return f"Pengembalian untuk {nama} setelah {tahun} tahun: Rp{total:,.2f}"
        return f"Anggota {nama} tidak ditemukan."

Contoh Aksi:

Pengembalian untuk Ihsan setelah 2 tahun: Rp5,500,000.00

Query 4: Membuat Class untuk Koperasi

Membuat class Koperasi yang mewarisi UMKMSystem dengan atribut tambahan transaksi. Class ini menggunakan inheritance untuk mewarisi atribut dan method dari UMKMSystem. Atribut transaksi adalah list kosong untuk menyimpan transaksi.

Tip Kunci:

Do: Selalu gunakan super().__init__ untuk menginisialisasi kelas induk.

Don't: Hindari mendefinisikan ulang atribut yang sudah ada di kelas induk. Ini akan menyebabkan redundansi dan potensi masalah data.


class Koperasi(UMKMSystem):
    def __init__(self, nama_umkm):
        super().__init__(nama_umkm)
        self.transaksi = []

Hasil: Koperasi kini memiliki semua fitur UMKMSystem ditambah dengan pencatatan transaksi yang spesifik.

Query 5: Method untuk Mencatat Transaksi Koperasi

Metode ini dirancang untuk merekam setiap aktivitas "beli" atau "jual" anggota, mencatatnya sebagai dictionary yang terstruktur dalam daftar transaksi.

Tip Kunci:

Do: Pastikan Anda memvalidasi jenis transaksi yang dimasukkan (hanya "beli" atau "jual").

Don't: Jangan pernah lupa mencatat nama anggota untuk setiap transaksi


    def catat_transaksi(self, nama, jenis_transaksi, jumlah):
        transaksi = {
            "nama": nama,
            "jenis": jenis_transaksi,
            "jumlah": jumlah
        }
        self.transaksi.append(transaksi)
        print(f"Transaksi {jenis_transaksi} Rp{jumlah:,} untuk {nama} dicatat.")

Contoh Aksi:

Transaksi jual Rp10,000,000 untuk Ihsan dicatat.

Query 6: Method untuk Menghitung Keuntungan Koperasi

Menambahkan method hitung_keuntungan untuk menghitung keuntungan (jual - beli).

Tip Kunci:

Do: Gunakan .lower() pada jenis transaksi untuk menangani input yang case-insensitive, membuat sistem lebih fleksibel.

Don't: Jangan berasumsi semua transaksi valid. Selalu validasi jenis transaksi sebelum menghitung.


    def hitung_keuntungan(self):
        total_jual = 0
        total_beli = 0
        for transaksi in self.transaksi:
            if transaksi["jenis"].lower() == "jual":
                total_jual += transaksi["jumlah"]
            elif transaksi["jenis"].lower() == "beli":
                total_beli += transaksi["jumlah"]
        keuntungan = total_jual - total_beli
        return f"Keuntungan koperasi: Rp{keuntungan:,}"

Contoh Aksi:

Keuntungan koperasi: Rp5,000,000

Query 7: Membuat Class untuk Bank Sampah

Membuat class BankSampah yang mewarisi UMKMSystem dengan atribut data_sampah.

Tip Kunci:

Do: Rancang struktur data_sampah dengan sangat jelas. Ini akan mempermudah akses ke nilai tukar dan jumlah sampah per anggota.

Don't: Jangan gunakan list untuk data sampah karena sulit mengelompokkan per jenis.


class BankSampah(UMKMSystem):
    def __init__(self, nama_umkm):
        super().__init__(nama_umkm)
        self.data_sampah = {
            "plastik": {"nilai_tukar": 5000, "jumlah_kg": {}},
            "kertas": {"nilai_tukar": 2000, "jumlah_kg": {}}
        }

Hasil: BankSampah kini siap mengelola kontribusi sampah dengan efisien.

Query 8: Method untuk Mencatat Data Sampah

Menambahkan method catat_sampah untuk mencatat jenis dan jumlah sampah per anggota.Method ini memperbarui jumlah_kg di data_sampah berdasarkan nama dan jenis_sampah

Tip Kunci:

Do: Selalu cek apakah jenis sampah yang dimasukkan valid sebelum mencatatnya. Ini penting untuk mencegah data yang tidak konsisten.

Don't: Jangan pernah langsung menambahkan data tanpa memeriksa apakah anggota sudah ada di dalam dictionary


    def catat_sampah(self, nama, jenis_sampah, jumlah_kg):
        if jenis_sampah in self.data_sampah:
            if nama in self.data_sampah[jenis_sampah]["jumlah_kg"]:
                self.data_sampah[jenis_sampah]["jumlah_kg"][nama] += jumlah_kg
            else:
                self.data_sampah[jenis_sampah]["jumlah_kg"][nama] = jumlah_kg
            print(f"{jumlah_kg} kg {jenis_sampah} dari {nama} dicatat.")
        else:
            print(f"Jenis sampah {jenis_sampah} tidak dikenal.")

Contoh Aksi:

5 kg plastik dari Ihsan dicatat.

Query 9: Method untuk Menghitung Nilai Tukar Sampah

Menambahkan method hitung_nilai_tukar untuk menghitung nilai rupiah dari sampah. Method ini mengalikan jumlah sampah per jenis dengan nilai tukar dan menjumlahkan totalnya

Tip Kunci:

Do: Gunakan .get(nama, 0) saat mengambil jumlah kilogram sampah dari anggota. Ini mencegah error jika anggota tidak ada dalam catatan untuk jenis sampah tertentu.

Don't: Jangan berasumsi setiap anggota memiliki data sampah untuk setiap jenis.


    def hitung_nilai_tukar(self, nama):
        total_nilai = 0
        for jenis_sampah, data in self.data_sampah.items():
            jumlah_kg = data["jumlah_kg"].get(nama, 0)
            nilai = jumlah_kg * data["nilai_tukar"]
            total_nilai += nilai
        return f"Nilai tukar sampah untuk {nama}: Rp{total_nilai:,}"

Contoh Aksi:

Nilai tukar sampah untuk Ihsan: Rp25,000

Query 10: Method untuk Pesan Edukasi Lingkungan

Menambahkan method pesan_edukasi untuk memberikan pesan berdasarkan jumlah sampah.Method ini menghitung total sampah anggota dan mengembalikan pesan berbeda menggunakan if-elif-else. Pesan menggunakan string.

Tip Kunci:

Do: Gunakan kondisi yang jelas dan terpisah untuk setiap rentang jumlah sampah.

Don't: Hindari pesan yang terlalu panjang atau membingungkan; jaga agar tetap singkat, padat, dan berdampak.


    def pesan_edukasi(self, nama):
        total_sampah = 0
        for jenis_sampah in self.data_sampah:
            total_sampah += self.data_sampah[jenis_sampah]["jumlah_kg"].get(nama, 0)
        
        if total_sampah > 50:
            return "Selamat! Anda telah membantu mengurangi sampah secara signifikan. Mari terus dukung daur ulang untuk bumi yang lebih hijau!"
        elif total_sampah > 20:
            return "Kerja bagus! Anda berkontribusi pada lingkungan yang lebih bersih. Ayo kumpulkan lebih banyak sampah untuk didaur ulang!"
        elif total_sampah > 0:
            return "Terima kasih telah berpartisipasi di bank sampah. Setiap kilogram sampah yang Anda kumpulkan membantu lingkungan!"
        else:
            return "Ayo mulai kumpulkan sampah untuk mendukung keberlanjutan lingkungan!"

Contoh Aksi:

Terima kasih telah berpartisipasi di bank sampah. Setiap kilogram sampah yang Anda kumpulkan membantu lingkungan!

Query 11: Mengintegrasikan Semua Komponen

Program utama membuat objek Koperasi dan BankSampah, meminta input pengguna untuk anggota, transaksi, dan sampah, lalu menampilkan laporan lengkap

Tip Kunci:

Do: Gunakan loop untuk input berulang agar program lebih interaktif. Pastikan untuk menambahkan penanganan error (try-except) untuk input numerik agar program tidak crash.

Don't: Jangan lupakan validasi input! Ini adalah langkah krusial untuk mencegah kesalahan runtime dan memastikan data yang masuk akurat.


def main():
    print("=== Sistem Pengelolaan UMKM, Koperasi, dan Bank Sampah ===")
    
    nama_umkm = input("Masukkan nama UMKM: ")
    koperasi = Koperasi(nama_umkm)
    bank_sampah = BankSampah(nama_umkm)
    
    while True:
        nama = input("\nMasukkan nama anggota (atau 'selesai' untuk lanjut): ")
        if nama.lower() == 'selesai':
            break
        try:
            pinjaman = float(input("Masukkan jumlah pinjaman (Rp): "))
            koperasi.tambah_anggota(nama, pinjaman)
            bank_sampah.tambah_anggota(nama, pinjaman)
        except ValueError:
            print("Input jumlah pinjaman tidak valid. Harap masukkan angka.")
    
    nama = input("\nMasukkan nama anggota untuk cek pengembalian: ")
    try:
        tahun = int(input("Masukkan jumlah tahun: "))
        print(koperasi.hitung_pengembalian(nama, tahun))
    except ValueError:
        print("Input jumlah tahun tidak valid. Harap masukkan angka.")
    
    while True:
        nama = input("\nMasukkan nama anggota untuk transaksi (atau 'selesai' untuk lanjut): ")
        if nama.lower() == 'selesai':
            break
        jenis = input("Masukkan jenis transaksi (beli/jual): ")
        try:
            jumlah = float(input("Masukkan jumlah transaksi (Rp): "))
            koperasi.catat_transaksi(nama, jenis, jumlah)
        except ValueError:
            print("Input jumlah transaksi tidak valid. Harap masukkan angka.")
    
    print(koperasi.hitung_keuntungan())
    
    while True:
        nama = input("\nMasukkan nama anggota untuk sampah (atau 'selesai' untuk lanjut): ")
        if nama.lower() == 'selesai':
            break
        jenis_sampah = input("Masukkan jenis sampah (plastik/kertas): ")
        try:
            jumlah_kg = float(input("Masukkan jumlah sampah (kg): "))
            bank_sampah.catat_sampah(nama, jenis_sampah, jumlah_kg)
        except ValueError:
            print("Input jumlah sampah tidak valid. Harap masukkan angka.")
    
    nama = input("\nMasukkan nama anggota untuk cek nilai tukar dan edukasi: ")
    print(bank_sampah.hitung_nilai_tukar(nama))
    print(bank_sampah.pesan_edukasi(nama))

if __name__ == "__main__":
    main()
